### Model
1. 會議
2. 與會人員(包含子類別)
3. 臨時動議
4. 報告事項
5. 討論事項
6. 決議
7. 附件

### 基本功能
1. 與會人員可以參加會議
2. 與會人員可以查詢、觀看自己的會議紀錄
3. 與會人員可以提出修改會議的請求
4. 會議可以附加附件
5. 與會人員可以提出臨時動議
6. 追蹤決議執行結果

### 額外功能
- [ ] 開會時寄信給與會人員
- [ ] 引導用的聊天機器人
- [ ] 黑暗模式(也許會當成預設主題)
- [ ] 網頁顯示目前正在進行的會議
- [ ] 響應式網頁
- [ ] 已結束的會議會顯示在首頁
